语音降噪

基于深度学习的实时语音增强技术，有效抑制环境噪声，提升通话与录音清晰度。

核心技术

  • 采用 RNNoise + DeepFilterNet 混合模型，兼顾低延迟与高降噪精度
  • 支持稳态噪声（空调/风扇）与非稳态噪声（键盘/人声干扰）双重抑制
  • 端到端处理延迟 < 10ms，CPU 占用率 < 5%

应用场景

  • 视频会议 / 在线课堂 / 直播推流
  • 智能客服通话录音质检
  • 车载语音交互前处理

回声消除

精准消除扬声器播放声音被麦克风再次采集产生的回声，保障双工通话体验。

技术原理

  • 自适应滤波器估计回声路径，从近端信号中减去远端参考信号的副本
  • 结合非线性残余回声抑制，解决双讲场景下的回声泄露问题
  • 支持 16kHz / 48kHz 多采样率，适配各类音频硬件

性能指标

  • ERLE（回声回损增强）≥ 30dB
  • 双讲透明度 > 95%，不影响正常对话

语音唤醒

低功耗关键词检测技术，让设备随时待命响应自定义唤醒词。

产品特性

  • 支持自定义唤醒词，3~8 字中文/英文均可训练
  • 误唤醒率 < 1次/24小时，拒识率 < 1%
  • 模型大小 < 500KB，可在 MCU / DSP 等嵌入式芯片上运行

典型应用

  • 智能音箱 / 智能家居面板
  • 车载语音助手
  • 可穿戴设备免提操控

语音识别

高精度语音转文字服务，支持实时流式与离线文件转写。

能力概览

  • 普通话识别准确率 ≥ 98%（安静环境）
  • 支持中英混合、方言（粤语/四川话等）识别
  • 实时流式响应时间 < 300ms，支持标点智能预测

接入方式

  • WebSocket 实时流式接口
  • RESTful 离线文件转写接口
  • Android / iOS / Windows SDK

语音合成

自然流畅的文字转语音服务，提供多种音色与情感风格选择。

音色库

  • 预置 20+ 高品质音色：新闻播报 / 温柔女声 / 活力男声 / 童声等
  • 支持声音克隆，仅需 5 分钟录音即可定制专属音色
  • 支持 SSML 标记语言，精细控制语速、停顿、重音

技术指标

  • MOS 评分 ≥ 4.5（满分5分）
  • 首包延迟 < 200ms，支持流式播放

音乐分离

AI 驱动的音源分离技术，将混合音频拆分为人声、伴奏、鼓点等独立轨道。

分离能力

  • 支持 4-Stem 分离：人声 / 贝斯 / 鼓 / 其他乐器
  • 支持 2-Stem 快速分离：纯人声 / 纯伴奏
  • 分离后音质损失 < 3dB，无明显相位失真

使用场景

  • K歌 App 伴奏提取
  • 音乐制作采样 / Remix
  • 影视后期对白提取
